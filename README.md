# Nutrition Tracker

## Description
A simple Flutter app to track daily calorie, protein, carbs, and fats intake. It allows users to add meals and view their nutritional summary for the day, and set goals. Users can view their progress and trends based on what they've done previously.

## Current Features
- Add meals with calorie information to track daily intake.
- View total calories consumed for the day.
- Set, edit, and track daily goals for water and food intake.
- A food dictionary for preloading foods eaten a lot. Users can quickly add all values without needing to recalculate or reenter values.
- A calendar with all previously entered daily intake and goals.

## How to Use
1. Pull the code
2. Run `flutter pub get` to make the code run on your system
3. Run `flutter run` to start the app. Follow the prompts to select where you'd like it to display
- Note: This method only runs in debug mode. You'll need to install it onto your smartphone if you would like to use it there. There is no current app download on any appstore.
